A modern, dark-mode Todo application built with **React**, **Vite**, and **Tailwind CSS**.  
Tasks are persisted in **localStorage**, so they survive refreshes and browser restarts.

## Features

- **Add / toggle / delete tasks**
- **Filter** by All / Active / Completed
- **Clear completed** tasks in one click
- **Dark / light mode toggle** (saved to localStorage)
- **LocalStorage persistence** for todos and theme
- Responsive, glassmorphism-inspired UI

## Getting started

```bash
cd my-ai-project
npm install
npm run dev
```
